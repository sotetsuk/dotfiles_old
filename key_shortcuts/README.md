# Key Shortcuts

使えてほしい想定ショートカットの一覧をまとめる。
環境毎 (Mac/Ubuntu) にこれらを実現できるように設定をする。

## 修飾キー

- `Ctrl-`: Aの右隣
- `LCmd-`: スペース左隣
- `LOpt-`: `LCmd` 左隣
- `RCmd-`: スペース右隣
